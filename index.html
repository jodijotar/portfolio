    Agile Krakens - FATEC

    Setembro de 2024 - Novembro de 2024

    Atuei como Scrum Master e no desenvolvimento do backend de uma aplicação web de consulta, tratamento e interpretação de dados públicos da atuação dos vereadores do último mandato de São José dos Campos. Meu trabalho enquanto Scrum Master enquadra, principalmente:
    - Revisão da Arquitetura de Software com o P.O através da enumeração de requisitos por User Stories (Product Backlog).
    - Gerenciar cerimônias baseadas em metodologia ágil, com objetivo de garantir o MVP de cada sprint, alinhadamente com a capacidade/tempo do DevTeam.
    - Diferentemente de um contexto corporativo, a atuação do Scrum Master nesse cenário acadêmico abrange não somente Engenharia de Software, mas também minha contribuição com código.


    Desafios encontrados:

    Minha contribuição como desenvolvedor pode ser ilustrada por minha abordagem diante da necessidade de coletar dados públicos que são constantemente atualizados. Diante esse desafio, procurei tecnologias e recursos para coletar, tratar, inserir e exibir esses dados de forma totalmente dinâmica e automatizada. Portanto, para atingir este objetivo utilizei essas tecnologias:
    - Web Scraping: BeautifulSoup e requests (Bibliotecas de Python) para coletar dados que seriam facilmente obtidos através de apenas uma requisição; Selenium para dados que necessitavam de navegação e interação com outros componentes para serem obtidos.
    - Tratamento dos dados: durante a coleta dos dados, formatei os campos desejados utilizando Expressões Regulares (Regex)
    - Inserção dos dados na minha Database: após o script de scraping ser executado, arquivos JSON eram gerados, e também, outro modulo realizava a inserção automaticamente dos dados no SGBD
    - Exibição: a geração do site como um todo foi feita de forma totalmente dinâmica, através de rotas, a aplicação é montada com base no meu banco de dados. Portanto, com apenas um comando eu atingi o objetivo de atualizar não só a minha database inteira com as informações atuais da câmara dos vereadores da minha cidade, mas também exibi-las no site de forma totalmente automatizada.
